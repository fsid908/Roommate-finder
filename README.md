Roommate Finder

A full-stack roommate matching web application built with **React (Vite)** and **Node.js + Express**.  
The platform allows users to create profiles, browse available listings, and connect with potential roommates.

Tech Stack

**Frontend**
- React
- Vite
- React Router
- CSS

**Backend**
- Node.js
- Express.js
- MongoDB


Project Structure

roommate-finder/
│
├── frontend/   → React + Vite client
├── backend/    → Express API server

---

Installation & Setup

Clone the repository

```bash
git clone https://github.com/your-username/roommate-finder.git
cd roommate-finder
```

Setup Frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend will run on:
```
http://localhost:5173
```

Setup Backend

```bash
cd backend
npm install
npm start
```

Backend will run on:
```
http://localhost:5000
```


Features

- User profile creation
- Add and browse roommate listings
- Search and filter functionality
- REST API integration
- Responsive UI design
